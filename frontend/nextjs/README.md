## Getting Started
this project uses

add lucide.dev  
```bash
npm install lucide  
npm install lucide-react  
```

add axios  
```bash
npm install axios  
```

run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
